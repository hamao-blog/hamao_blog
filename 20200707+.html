php elseif

まーこれは簡単にいうと　
細かい条件設定ですね。

前回のイチゴの例で例えると

$x = イチゴの値段;　としましょう。

if ($x < 100){
echo 'すぐかえ！'
}
elseif($x < 150 ){
echo 'まあ、買ってもいいけど？'
}
　else{
echo '買うな！'
}

てな具合だと認識しています。

・・・・間違ってたらごめんなさい。笑
