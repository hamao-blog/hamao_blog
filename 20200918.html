swap領域 


今回はlaravelのFormファサードを使いたくて嵌った内容をしるす

まずlaravelにはformファサードというとっても便利な機能があります
まあよくわかんないけど
まずcomposerからインストールするのですが
composer require "laravelcollective/html":"6.*"
これやるだけ
が。しかしここではまった
バージョンが合ってなかったりした
後、
mmap() failed: [12] Cannot allocate memory

mmap() failed: [12] Cannot allocate memory
こんなん出て調べたらどうもメモリ不足ってことらしい
swap領域を増やしちゃお
まず確認
free -m
swapが0なら
sudo dd if=/dev/zero of=/swapfile bs=1M count=2048
sudo chmod 600 /swapfile
sudo mkswap /swapfile
sudo swapon /swapfile

ddコマンドで指定サイズの空のファイルを作成
mkswapで、作成したファイルをスワップ領域として使用できるようにする
swaponでスワップ領域を有効化

ではまた！
